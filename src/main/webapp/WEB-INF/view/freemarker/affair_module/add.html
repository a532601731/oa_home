面板标题
本模板基于Bootstrap框架实现，支持所有Bootstrap插件。
模板中CSS动画效果基于Animate.CSS框架和WOW.Js框架实现，支持所有Animate.CSS中动画效果。
模板中所有Dom监听事件基于JQuery框架实现，支持所有JQuery插件，当前版本jquery-1.8.3。
